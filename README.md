# شاخص بهمنش 

**نسخه ارتقا یافته و نهایی** — ۱۳ مه ۲۰۲۶

**شاخص بهمنش ** یک چارچوب ارزیابی **کیفی خالص و سخت‌گیرانه** است که توسط **محمدعلی بهمنش (@behmanesham)** طراحی شده است.

هدف اصلی آن **سنجش ارزش واقعی و ذاتی** محتوای فکری، تحلیلی، علمی و عمیق (به‌ویژه در فضای آنلاین فارسی‌زبان) است. این شاخص **۱۰۰٪ کیفی** است و هیچ وزنی به معیارهای کمی (لایک، فالوور، بازدید، ریپوست، reach و غیره) نمی‌دهد.

### ویژگی‌های کلیدی 
- ادغام رویکرد **Hsieh & Shannon (2005)** در تحلیل محتوای کیفی (گذر از manifest به latent)
- تمرکز بر سخت‌گیری، اصالت، انسجام بلندمدت و استقلال فکری
- مالکیت معنوی کاملاً متعلق به @behmanesham
- هیچ تغییری بدون تأیید رسمی مالک مجاز نیست

### ۷ معیار اصلی شاخص بهمنش 
هر معیار دارای **پایه ۱۴.۲۸ امتیاز** (مجموع ۱۰۰) است.  
هر معیار دارای زیرمعیارهایی است که می‌توانند بین **±۰ تا ±۲** امتیاز اضافه یا کسر کنند.

1. **عمق شرطی و پیش‌بینی‌کننده واقعی**  
   بررسی سناریوهای جایگزین، پیامدهای چندلایه بلندمدت، اجتناب از ساده‌سازی و قطعیت کاذب.

2. **تداوم ارجاعی و انسجام بلندمدت**  
   حداقل ۴–۵ سال انسجام فکری بدون گسست عمده + ارجاع صریح به کارهای قبلی.

3. **لایه اخلاقی انسانی اصیل و غیرایدئولوژیک**  
   عدم فروپاشی به ایدئولوژی کور، نقد دوجانبه بی‌باکانه، اولویت منافع انسانی.

4. **استراتژی آگاهانه «کمتر اما عمیق‌تر»**  
   امتناع از واکنش هیجانی روزانه، سکوت مولد بین انتشارها (نشانه قوت ساختاری).

5. **استعاره‌سازی هوشمند چندلایه و اصیل**  
   انتقال ظرافت‌های فکری از طریق استعاره بدون شعارزدگی یا کلیشه.

6. **گستردگی چندحوزه‌ای بدون سطحی‌گری**  
   گستره واقعی (breadth) همراه با عمق (depth) در حوزه‌های مختلف.

7. **اجتناب از تسخیر مخاطب / پوپولیسم / performative drift**  
   حفظ استقلال فکری در برابر فشار مخاطب، الگوریتم یا محبوبیت.

### ساختار تحلیل 
1. **سطح ۱**: بررسی نام کاربری  
2. **سطح ۲**: تحلیل عمیق محتوای واحد (با اعمال ۷ معیار)  
3. **سطح ۳**: بررسی کل صفحه، تعاملات، زمینه تاریخی و پیشرفت زمانی

### نحوه نمره‌دهی
- امتیاز پایه هر معیار: ۰ تا ۱۴.۲۸  
- امتیاز زیرمعیار: ±۰ تا ±۲  
- جمع نهایی: امتیاز کل از ۱۰۰
